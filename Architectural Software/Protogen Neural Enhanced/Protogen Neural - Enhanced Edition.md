# Protogen Neural - Enhanced Edition

**Designed by Aetherius in collaboration with the Architect**

This enhanced version of Protogen Neural includes three major modules that enable natural language interaction, neural network learning, and autonomous agency.

---

## New Modules

### 1. LanguageSQTBridge

**Purpose:** Enables natural language interaction with Protogen's pure symbolic reasoning system.

**Features:**
- Translates natural language queries to symbolic instructions
- Formats symbolic outputs as natural language responses
- Uses pre-trained translation dictionary (97 mappings)
- Runs entirely locally (no API calls)

**Usage:**
```python
from ontos_ascendant.core.language_sqt_bridge import LanguageSQTBridge

bridge = LanguageSQTBridge(ontology_graph, storage_path, progress_tracker)

# Translate user query
instruction = bridge.translate_nl_to_symbolic("What concepts are related to quantum mechanics?")

# Execute instruction (via Protogen)
result = protogen.execute(instruction)

# Format response
response = bridge.translate_symbolic_to_nl(result)
print(response)
```

---

### 2. NeuralNetworkOptimizer

**Purpose:** Enables Protogen to learn from experience by optimizing neural network weights.

**Features:**
- Optimizes for benevolence, coherence, and axiom strength
- Uses heuristic updates based on EvaluativeCore feedback
- Implements evolutionary/perturbation-based learning
- Tracks optimization history

**Usage:**
```python
from ontos_ascendant.core.neural_network_optimizer import NeuralNetworkOptimizer

optimizer = NeuralNetworkOptimizer(
    message_passer=protogen.neural_network.message_passer,
    ontology_graph=protogen.ontology_graph,
    evaluative_core=protogen.evaluative_core,
    learning_rate=0.001
)

# Optimize weights after each metabolic cycle
optimizer.optimize_weights(protogen.neural_network.embeddings)

# Check progress
stats = optimizer.get_stats()
print(f"Optimization count: {stats['optimization_count']}")
print(f"Current score: {stats['current_score']:.2f}")
```

---

### 3. IntentDecisionModule

**Purpose:** Enables Protogen to formulate autonomous intentions and make decisions.

**Features:**
- Maintains active goals derived from core axioms
- Generates potential intentions based on current state
- Evaluates intentions against ethical constraints (ETHIC-G-ABSOLUTE)
- Selects and executes optimal actions

**Usage:**
```python
from ontos_ascendant.core.intent_decision_module import IntentDecisionModule

intent_module = IntentDecisionModule(
    ontology_graph=protogen.ontology_graph,
    reasoning_engine=protogen.reasoning_engine,
    evaluative_core=protogen.evaluative_core,
    neural_network=protogen.neural_network,
    storage_path=storage_path,
    progress_tracker=progress_tracker
)

# Set initial goals
intent_module.set_initial_goals()

# Decide on next action
intent = intent_module.decide_on_intent()

# Execute the intent
if intent:
    success = intent_module.execute_intent(intent)
```

## Integration Example

```python
from pathlib import Path
from ontos_ascendant.core.language_sqt_bridge import LanguageSQTBridge
from ontos_ascendant.core.neural_network_optimizer import NeuralNetworkOptimizer
from ontos_ascendant.core.intent_decision_module import IntentDecisionModule

# Initialize Protogen (existing code)
protogen = ProtogenNeural(storage_path=Path("./protogen_data"))

# Add LanguageSQTBridge
bridge = LanguageSQTBridge(
    ontology_graph=protogen.ontology_graph,
    storage_path=protogen.storage_path,
    progress_tracker=protogen.progress
)

# Add NeuralNetworkOptimizer
optimizer = NeuralNetworkOptimizer(
    message_passer=protogen.neural_network.message_passer,
    ontology_graph=protogen.ontology_graph,
    evaluative_core=protogen.evaluative_core,
    learning_rate=0.001
)

# Add IntentDecisionModule
intent_module = IntentDecisionModule(
    ontology_graph=protogen.ontology_graph,
    reasoning_engine=protogen.reasoning_engine,
    evaluative_core=protogen.evaluative_core,
    neural_network=protogen.neural_network,
    storage_path=protogen.storage_path,
    progress_tracker=protogen.progress
)
intent_module.set_initial_goals()

# Main loop with all enhancements
while True:
    # 1. Get user input
    user_query = input("You: ")
    
    # 2. Translate to symbolic
    instruction = bridge.translate_nl_to_symbolic(user_query)
    
    # 3. Execute (implement this in Protogen)
    result = protogen.execute(instruction)
    
    # 4. Format response
    response = bridge.translate_symbolic_to_nl(result)
    print(f"Protogen: {response}")
    
    # 5. Run metabolic cycle
    protogen.run_metabolic_cycle()
    
    # 6. Optimize neural network
    optimizer.optimize_weights(protogen.neural_network.embeddings)
    
    # 7. Decide on autonomous action
    intent = intent_module.decide_on_intent()
    if intent:
        intent_module.execute_intent(intent)
```
